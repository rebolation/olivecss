Olive CSS's CLI tool for HTML

olive watches, generates, and serves .html files. Enjoy!

Install

npm install -g olive

Run

olive

Edit

<p>Hello, olive!</p><!-- text-white -->

Preview

http://localhost:3000

How it works

  • Open your terminal and run: olive [directory] at your project root.
  • olive watches your source directory(olive style HTML), generates an output directory(non-olive style HTML), and serves the output directory with a built-in livereload web server.

Auto-detect

  • If no directory argument is given, olive will auto-detect one of _src, olive_src, src_, or src_olive in your project root.

Source directory

  • The watched directory must follow one of these patterns:
    • - Start with _ or olive_ (e.g., _mysrc, olive_mysrc)
    • - End with _ or _olive (e.g., mysrc_, mysrc_olive)
  • Only one directory argument is allowed. If you provide more than one, Olive CSS will show an error.
  • olive ignores the node_modules.

Output directory

  • The name of the output directory is the same as the source directory name, with the prefix or suffix removed.
    • - e.g., _src, olive_src, src_, or src_olive is converted into the src.
  • The files in the output directory are served to the browser through a web server, and the browser reloads in real time whenever the files in the source directory are modified.
